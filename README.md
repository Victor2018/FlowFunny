	本项目是开发者在工作过程中的所用到的一些技术总结，和一些开源项目的使用，旨在提高开发效率走敏捷开发路线

# 项目名字：
	FlowFunny 就是希望开发者能开心，快乐的学习Android并从中获得趣味,搞笑我们是认真的！
